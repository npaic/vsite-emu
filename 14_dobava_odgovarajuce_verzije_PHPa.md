# Dobava PHP-a #


potrebna verzija: >5.x


PHP se dobavlja na način da u konzoli upišemo prvu naredbu:

`sudo apt-get install php5`

nakon izvršavanja upisujemo drugu naredbu:

`sudo apt-get install libapache-mod-php5`
