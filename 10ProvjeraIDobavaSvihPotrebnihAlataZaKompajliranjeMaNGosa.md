Potrebni alati su:

•	GCC 4.1.X ( gcc / g++ / cpp / fort77 / g77 / gpp) <br>
•	automake <br>
•	autoconf <br>
•	glibc & glibc-dev(glibc headers) <a href='.md'>libc6 </a> <br>
•	make <br>
•	mysql-server 4.1 or mysql-server 5.0 && libmysql++-dev <br>
•	libtool <br>
•	OpenSSL (libssl-dev) <br>
•	subversion and patch <br>
•	git <br>
•	zlibc <br>

Mogu se nabaviti upisivanjem sljedeceg u Terminal:<br>
<br>
<b>sudo apt-get install gcc g++ automake autoconf make libmysql++-dev libtool libssl-dev subversion patch zlibc libc6 git git-core pkg-config</b>
